# چطور یک نوشته جدید ایجاد کنیم
<!-- position: 2 -->

وارد پنل مدیریت شوید و از نوار سمت راست بر روی نوشته جدید کلیک کنید.


1. عنوان را بنویسید.
2. محتوا را بنویسید.
3. بر روی دکمه **ذخیره** کلیک کنید.
4. انجام شد!
